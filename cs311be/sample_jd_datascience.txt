MÔ TẢ CÔNG VIỆC:
Chúng tôi đang tìm kiếm một Data Scientist tài năng để tham gia vào đội ngũ AI/ML của chúng tôi. Bạn sẽ chịu trách nhiệm phát triển các mô hình machine learning, phân tích dữ liệu phức tạp và tạo ra các giải pháp AI sáng tạo để giải quyết các thách thức kinh doanh thực tế.

TRÁCH NHIỆM CHÍNH:
• Phát triển và triển khai các mô hình machine learning và deep learning
• Phân tích dữ liệu lớn để khám phá insights và patterns
• Thiết kế và thực hiện các thí nghiệm A/B testing
• Xây dựng pipeline xử lý dữ liệu và feature engineering
• Tối ưu hóa hiệu suất mô hình và monitoring
• Hợp tác với các team khác để tích hợp AI solutions
• Viết báo cáo kỹ thuật và trình bày kết quả cho stakeholders
• Nghiên cứu và áp dụng các công nghệ AI/ML mới nhất

YÊU CẦU KỸ NĂNG:

Kỹ năng bắt buộc:
• Thành thạo Python và các thư viện ML (scikit-learn, TensorFlow, PyTorch)
• Kinh nghiệm với SQL và NoSQL databases
• Kiến thức vững chắc về statistics và probability
• Kinh nghiệm với data visualization (Matplotlib, Seaborn, Plotly)
• Hiểu biết về machine learning algorithms (classification, regression, clustering)
• Kinh nghiệm với big data tools (Spark, Hadoop)
• Git version control và collaborative development

Kỹ năng ưu tiên:
• Kinh nghiệm với deep learning và neural networks
• Kiến thức về NLP và computer vision
• Kinh nghiệm với cloud platforms (AWS, GCP, Azure)
• Kinh nghiệm với MLOps và model deployment
• Kỹ năng communication và presentation
• Kinh nghiệm làm việc với real-time data streams
